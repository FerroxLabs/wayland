# Wayland

Wayland is a local-first desktop and headless AI agent command center. It can
run Wayland-Core directly and can drive supported agent CLIs such as Claude
Code, Codex, Gemini, Qwen, Goose, and others through their own authentication
and runtime surfaces.

This package directory is used by the Wayland distribution pipeline. For user
downloads, source builds, security notes, and contribution guidance, use the
repository-level documentation:

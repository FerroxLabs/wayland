# Filesystem setup

No setup required. After install, Wayland will prompt you to pick the
directories this MCP is allowed to read and write. The default is your home
folder; you can tighten the scope from the MCP settings.

You can change the allowed directories any time. The server only sees files
inside the directories you authorize.
